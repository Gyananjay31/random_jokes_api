# Random Jokes API

## Setup Instructions

1. Clone the repo
2. Run `npm install`
3. Start server with `node index.js`
4. Visit `http://localhost:3000/api/jokes/random` to get a random joke

## API Endpoints

- `GET /api/jokes/random` - Returns a random joke in JSON format
